gh secret

机密(Secrets) 可以在存储库、环境或组织级别设置，以便在 GitHub Actions中使用。用户的机密可以设置为，还能在 GitHub 代码空间中使用。运行gh help secret set学习如何开始。

Commands

  • gh secret list
  • gh secret remove
  • gh secret set

Options

-R, --repo <[HOST/]OWNER/REPO>
使用 [HOST/]OWNER/REPO 格式，选择另一存储库
