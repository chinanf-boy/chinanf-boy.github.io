Rust Compiler Performance Triage

  • 2020-11-24：1 个退步，2 个改进，2 个混合

这周我们看到了#79237的着陆，它本身没有成功，但为在 macOS 上支持 split debuginfo 打开了大门。这最终会是巨大的成功，因为我们可以避免重新收集 debuginfo 的同时，保留对 lldb 和 Rust backtraces 的支持。#79361跟踪 rustc 标志的稳定性，但尚未 100% 明确，向稳定版本的用户推出该产品。

分类由 @jyn514 和 @simulacrum 完成。
