本周引用句

UB（未定义行为） 的特殊之处在于它会毁坏你审查 bug 的能力，就像一种攻击免疫系统的疾病。未定义的行为可能具有任意的、非局部的，甚至非关联的邪恶魔法，从而破坏程序的确定性。这是不可容忍的，这就是为什么 safe Rust 对未定义的行为排查，是如此重要，尽管仍然存在 Rust 无法消除的漏洞类别。

– @trentj on rust-users
