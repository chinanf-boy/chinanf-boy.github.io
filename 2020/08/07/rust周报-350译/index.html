326 个请求在上周合并

  • 在裸函数参数上，压制 debuginfo
  • 将ParamEnv转换到Reveal::All时，规范化所有不透明类型
  • 在为大型 if 表达式进行类型检查和构建 MIR 时，确保栈
  • 用一个 iterative 的算法代替递归算法
  • 修复：给 trait objects 的#[track_caller]垫片
  • 让闭包和 generators 成为must_use类型
  • BTreeMap::drain_filter在迭代期间，不应该接触 root
  • 添加：str::(r)split_once
  • 添加：Vec::spare_capacity_mut
  • 添加：slice::array_chunks
  • 稳定化：const_type_id
  • 稳定化：Vec::leak作为一种方法
  • 稳定化：Result::as_deref和as_deref_mut
  • 让Option::unwrap为不稳定 const
  • 让mem::size_of_val和mem::align_of_val为不稳定 const
  • backtrace-rs：包括源代码列号（如果可用）
