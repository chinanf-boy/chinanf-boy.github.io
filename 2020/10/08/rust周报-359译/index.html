Rust Compiler Performance Triage

  • 2020-10-05：1 个性能退化，2 个改进，1 个混合

安静的一周。在综合基准上进行了相当大的性能退化，并进行了一些小改进。

#77023是一个有趣的案例。它将 slice lengths 的不变量，在len函数内，编码为assume intrinsic。它似乎导致了较小的编译时速度降低，但是在check构建性能（用于生成代码质量的代理）没有提升。实际上，LLVM 文档 特别不建议 过度使用assume intrinsic 的情况，比如 不变量不太可能对 optimizer 有很大帮助的时候，似乎就是这情况。
