Rust Core 的更新

共有 273 个拉取请求在上周合并

  • 收缩ParamEnv，变为 16 个字节
  • 稳定化：const mem::forget
  • typeck：将类型信息添加到 projection 中
  • clippy：给浮点运算操作，来点精确 lints
  • 正确标记 match arm 的结束 span
  • 只允许，enum 上的repr(i128/u128)
  • 如果某类型没有实现DerefMut，就在侧边栏，隐藏来自 Deref 的&mut self方法
  • 仅在windows-msvc目标上添加 CFGuard
  • 添加：VecDeque::range*方法
  • 添加：read_exact_at和write_all_at到 WASI 的FileExt
  • clippy：新的 lint：match_like_matches_macro
  • 优化，is_ascii for str and [u8]
  • arch：在 avx512f set 中，添加f32与f64 unaligned（未对齐）的 stores（储存） and loads（加载）
  • hashbrown：添加HashSet::drain_filter方法
