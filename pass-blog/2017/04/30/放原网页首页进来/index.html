把完整的网页放进，hexo中

我做了一下步骤

在不影响，快捷命令

hexo g -d

1. 原 html文件 放进根目录 source

hexo 根据 source 生成静态文件夹.

所以，放source才能被 看到

2. 在 html 文件中 第一行 写下

---
layout: false
---

不能被模版化

3. 再把 css 之类 一起放进 source 就成

最好一个文件夹，用 打包工具。

4. 把我的以前，简介域名，更换到 home.html

index.html 是首选，以前 用 home.html

提示，如果想，整个demo 文件夹 单纯 copy 过去
