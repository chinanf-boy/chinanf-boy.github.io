用法：(会有点权限问题)

  • 在任何 GitHub 存储库页面中。
  • 只需双击您需要的项目即可。
  • 单击右下角的下载按钮。（文件路径权限)
  • 查看进度仪表板并等待浏览器触发下载器。
  • 获取 ZIP 文件。
