  • 稳定化： proc 宏生成macro_rules items
  • 对所有零大小的数组，从needs_drop返回false
  • 优化 LexicalResolve::expansion
  • 删除自定义PartialEq impls for LocalInternedString
  • 优化 BitIter
  • 优化 dropck
  • 更多符号清理
  • 避免expand_pattern()中，不必要的’土地’分配
  • 避免从TokenTree到TokenStream的不必要转换
  • expand：简化派生的扩展
  • Fix：约束 trait ，以找到方法的建议
  • 语法：为 intersection- / and-patterns p1 @ p2添加解析器恢复
  • 减少虚假的，未使用生命周期警告
  • 当唯一的区别是大小写时，请注意建议
  • 使用结构化建议，来限制范围
  • Fix: 零大小的未初始化框
  • 添加，重叠的 ranges 到无法到达的模式 lint，的检查
  • 为依赖图，使用更多细粒度的锁
  • Fix: canonicalize_const_var泄漏了推断变量
  • mir-opt：改进 SimplifyLocals 传递，以便可以删除未使用的 const
  • 使用显式泛型参数，改进 APIT 的错误消息
  • 删除不可访问的单元元组，比较 binop 的代码生成
  • 避免使用 ICE，当include!被 stdin 箱子使用时
  • impl AsRef<[T]> for List<T>
  • hashbrown：删除大量#[inline]注解
  • 在调试版本中，始终内联mem::{size_of，align_of
  • 避免CString::new中的，重新分配
  • BTreeSet的 symmetric_difference 和 union 优化
  • cargo：允许--all-features在虚拟工作区的根目录中
  • rustup 安装：添加--profile标志，以覆盖配置文件
